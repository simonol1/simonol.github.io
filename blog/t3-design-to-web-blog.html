Design to Web

What is a responsive site, why is it important?

A responsive site is one which adjusts it layout based on the size of the screen which the user is viewing it on. Responsive sites are important in this day and age because we view web pages on many different screen sizes and the users experience is very important for repeat visits.

What is a mobile first design?

A mobile first design is one which caters to the mobile user first. It is built with the mobile user in mind, then additional elements and features are added later.

What are frameworks and their pros and cons?

Frameworks provide the developer with a foundation to use. This foundation of code saves the developer time and money. The down side of using this is that the developer is less likely to fully understand the code and when it comes to making changes they may come up short. The developer can also be limited by the framework itself and options available to build within it.

What is a wireframe and why is important?

A wireframe is a 'blueprint' drawing of a web page. It provides the designer with a clearer picture of what the web page will look like and makes it easier for them to see the overall layout of the page before diving in to design.
