JS Scope

Explain to a non-tech friend: What is 'scope' and how does it work in Javascript?

The term 'scope' in Javascript refers to the set of values, variables and functions that you have access to. Each variable has either a global or local scope. If a scope is 'global' it can be used by multiple functions and is always declared outside of the function code. If the scope is 'local', it is declared within the function and is used only by the function.
