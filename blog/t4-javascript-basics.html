Javascript Basics

How does Javascript compare to HTML & CSS?

Javascript is a much more dynamic language which can interact with users. HTML is the content and CSS is the style on a web page. Generally anything which interacts with users on a webpage will be in Javascript.

Explain control flow and loops using an example from everyday life?

A control flow structure contains a statement which is executed under particular conditions and loops repeat their statements. For example: What time can I finish work today? If I finish work before 5 pm I will go for a run, if I finish between 5 and 7 pm I will go home and cook dinner, but if I finish work after 7 pm, I will get takeaways. This function could be: workFinishTime ().

Explain what functions are and why they are useful?

A function is a set of statements that calculates a value or performs a task. Functions are useful in Javascript because once a function is created it can be called (on) anytime in the scope it is defined without rewriting the code.

Explain the difference between assessing data from arrays and objects?

Arrays provide a way to store multiple pieces of information in a single thing. An array is basically a list of values: numbers, strings, boolean values or even other arrays. To assess arrays we use the bracket syntax: array [x]. Objects are properties and can be accessed using a key with brackets or dot notion. Arrays have order and objects do not.
