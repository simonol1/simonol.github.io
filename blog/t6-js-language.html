JS Language

Explain to the a non-tech friend: In the JavaScript language, describe all the use cases for each of:

Parentheses (): These are used to hold a functions parameter (which are the names listed in the function definition). They are also used to hold strings (which must sit in single or doubles quotes) and containing or grouping expressions. Parentheses are also used to call functions.

brackets []: These are used to hold a simple array. If they are displayed like var myarray = [] a new array is created.

braces {}: Braces are used to hold code in a function, key value arrays and objects/propoerties. These can also be used to create an empty object.

single quotes '': These can be used for a string. Strings are used to represent text in JavaScript. Single or double quotes can be used for a string, but they must match. ie: The starting quote must match the ending quote.

double quotes "": See above. In Javascript, both single and double quotes are acceptable. I find using the double quotes easier because in English we use double quotes when "quoting" someone in text. The other benefit is that if we want to write "it's" in a string we can write it like that. If the string is surrounded by single quotes we need to use an escaped version it\'s.
